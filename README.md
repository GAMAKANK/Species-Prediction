# Iris Flower  Classification

Iris flower are of three types - 

- iris  setosa
-  iris virginica
-  iris versicolor

The dataset is used to predict the type of iris flower based on its features.

The features used for to predict the  type of iris flower are - 
-  sepal length in cm
-   sepal width in cm
-   petal length in cm
-   petal width in cm
